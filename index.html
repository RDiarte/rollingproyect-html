Conociendo el HTML

Esta guia nos va a permitir conocer para que sirve y algunos comandos basicos de html

El HTML es un lenguaje usado para desarrollar la estructura de una página web, las siglas HTML significa Hypertext Markup Language o en español lenguaje marcado de hipertexto. Este lenguaje es codificado por los navegadores para que los usuarios puedan ver la información contenida en la página web
Para trabajar con HTML solo se necesita un editor de texto plano. Existen muchos programas de este tipo por ejemplo: bloc de notas, sublime text, aptana, brackets editor, visual code, atom, etc.

Semantica del HTML

Estructura del cuerpo de una pagina web


  1. Header: Esta seccion esta dedicada al encabezado. etiqueta header
  2. Nav: Barra de navegacion. Etiqueta nav
  3. Aside: Identifica una columna. Etiqueta aside
  4. Section: Nos muesta una seccion del sitio. Etiqueta section
  5. Article: Espacio para escribir un articulo. Etiqueta article
  6. Footer: Pie de pagina. Comando footer

Etiquetas HTML5

El lenguaje de marcado HTML se caracteriza por que se construye a partir de etiquetas (tags). Cuando decimos que HTML es un lenguaje de hipertexto significa que tiene texto oculto, y este se escribe dentro de etiquetas.
  1. Una etiqueta son palabras claves que encerramos con los signos < y >, dentro de estos signos escribimos el nombre de la etiqueta: < etiqueta>
  2. La mayoría de las etiquetas se deben escribir en par, es decir, una etiqueta de apertura y otra de cierre: < etiqueta> < /etiqueta>
  3. Si necesitan buscar información sobre el uso de una etiqueta pueden hacerlo con el nombre de tag HTML.

Estructura de una página web HTML5

La estructura de una página web HTML5 se construye de la siguiente manera:


  1. < !DOCTYPE html> Doctype es una etiqueta simple que no necesita cierre, su función es facilitar información del tipo de documento al servidor web que aloja la página, además es necesaria para la comunicación y entre el navegador y el servidor

  2. < html> es una etiqueta par, es la que va a encerrar todo el documento html.

  3. < head> esta etiqueta par, define la cabecera del documento y contiene información del mismo, por ejemplo metadatos, scripts, estilos, ubicación de estilos, título de la página entre otros datos. Todo lo que esté dentro de estas etiquetas no es visible para el usuario.

  4. < body> Es una etiqueta par y es conocida como el cuerpo del sitio web, todo lo que se escriba dentro de esta etiqueta será visible por los navegadores.

Estructura de la cabecera de un sitio web



A partir del proyecto anterior, veremos que componentes van dentro de la etiqueta < head> < /head>. < meta> Esta es una etiqueta simple, su función es añadir información de la página, los buscadores como google, consultan información de la etiqueta < meta> buscando coincidencias con lo que el usuario pretende encontrar. También se usa para agregar que tipo de idioma estamos trabajando en el lenguaje de marcado, para establecer esto las etiquetas poseen atributos.


< meta charset="utf-8"> Establece que nuestro sitio utiliza caracteres en español

esto es “utf-8”

< meta name="author" content="El contenido"> Establece el autor del sitio.

< meta name="description" content="Agregar una descripción del sitio"> Establece la descripción que es utilizada posteriormente por los buscadores como google.

< link rel="" Type="" href=""> establece el icono o favicon que tendrá nuestra web, el atributo href=”...” indica donde está ubicado ese icono.

< title> Estructura de la cabecera Indica el título que aparece en la pestaña del navegador al ingresar a la web.
